# biasqd-userstudy

User study extension for the BiasQD experiment

*Catalina Jaramillo, M Charity, Paul Squires, Julian Togelius*

## Installation

```
python -m venv .venv
pip install -r requirements.txt
python app.py
```


## Usage
Swipe left to reject an applicant, swipe right to accept.

Votes are recorded in [votes.db](votes.db) as a SQLite database
